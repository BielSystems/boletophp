1. DESCRIÇÃO

   Módulo escrito em PHP para a geração de boletos bancários de diversos bancos.

   1.1. Versão 1.x:
  
        É uma coleção de scripts em linguagem estruturada.
        https://github.com/BielSystems/boletophp/tree/1.x-dev
        
   1.2. Versão 2.x:
  
        É uma biblioteca em linguagem orientada a objetos.
        https://github.com/BielSystems/boletophp/tree/2.x-dev
     
        Documentação em
        https://github.com/BielSystems/boletophp/blob/2.x-dev/README.txt
